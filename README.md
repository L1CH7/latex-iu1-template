# BMSTU IU1 LaTeX Template

За основу взят репозиторий: https://github.com/Orianti/bmstu-latex-class.git
В основу взят XeLaTeX, а не pdfLaTeX как в оригинале, плюс специфичные правки ГОСТ для кафедры ИУ1.

## Конфигурация

Для сборки используется latexmk с конфигурацией из файла .latexmkrc.

Для работы с диаграммами используется mermaid-cli, который устанавливается через configure.sh.

## Сборка

```bash
latexmk
```
